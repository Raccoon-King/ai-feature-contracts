# GrabbyAI Brand Style Guide

## Brand Overview

GrabbyAI is a practical AI and office-productivity brand built around a friendly raccoon mascot named **Grabby**.

The brand should feel like a **calm, organized digital office assistant**. Grabby represents helping users manage files, workflows, and tasks in a simple and approachable way.

The visual identity blends:

- clean modern SaaS design
- friendly mascot branding
- light inspiration from classic Super Nintendo era UI aesthetics

This influence shows in bold readable typography, confident color blocks, and playful but structured visual balance.

The brand is **not futuristic sci-fi AI**.
It should feel grounded, helpful, and productive.

---

## Brand Personality

GrabbyAI should communicate:

- Helpful
- Organized
- Friendly
- Competent
- Calm
- Practical
- Structured
- Slightly playful but professional

The tone should feel like:

> "A smart office assistant that helps you stay organized."

Not:

- futuristic
- robotic
- chaotic
- overly childish

---

## Visual Direction

The visual style should remain:

- Flat vector illustration
- Clean shapes
- Clear outlines
- Readable at small sizes
- Friendly but professional
- Suitable for software UI and docs

Influence from **classic SNES-era design** means:

- bold clear shapes
- confident colors
- strong silhouettes
- readable typography
- minimal rendering

Avoid:

- painterly illustration
- heavy gradients
- 3D rendering
- glossy bubble text
- sci-fi AI imagery
- neon cyber aesthetics

---

## Mascot Guidelines

**Grabby the raccoon** is the core of the brand.

He represents organization, helpfulness, and getting work done.

### Core Mascot Traits

Grabby should remain visually consistent:

- Flat raccoon illustration
- Rounded body proportions
- Blue-gray body tones
- Dark raccoon mask markings
- White facial accents
- Friendly calm smile
- Thick navy outlines

Color tones should align with the brand palette.

### Preferred Mascot Activities

Grabby should appear doing helpful office tasks:

- Organizing files
- Working at a desk
- Using a laptop or desktop
- Holding documents
- Reviewing folders
- Drinking coffee
- Helping onboard users

### Mascot Rules

Always maintain:

- Flat vector style
- Minimal shading
- Simple facial features
- Clear silhouettes
- Consistent proportions

Never:

- turn Grabby into a robot
- add realistic fur
- over-detail shading
- add complex lighting
- distort proportions

Grabby should always feel like the **same character**.

---

## Logo System

### Primary Logo

The primary logo includes:

Grabby mascot above the wordmark.

Wordmark structure:

**GrabbyAI**

Style rules:

- Flat typography
- No bubble lettering
- No heavy gradients
- No star icons
- Clean readable shapes

Color usage:

- "Grabby" → Navy
- "AI" → Bright Blue
- Optional Yellow underline accent

The mascot sits above or beside the wordmark depending on layout.

### Secondary Logo

Wordmark only: **GrabbyAI**

Use when:

- space is limited
- iconography is unnecessary
- documentation headers
- UI navigation bars

### App Icon

Primary app icon: **GA monogram**

Design traits:

- Navy background
- White or blue GA letters
- Minimal styling
- Rounded square container

### Approved Compact Marks

Use only:

- GA monogram
- Grabby face icon

Do not introduce additional logo symbols.

### Logo Do Not Use

Avoid:

- stars or sparkles
- robot imagery
- overly cartoon fonts
- glossy bubble text
- alternate mascots
- complex shapes
- tech symbols replacing the mascot

Grabby and the wordmark should remain the brand anchors.

## Color Palette

Inspired by West Virginia tones but adjusted for uniqueness.

### Core Colors

| Color | Hex |
|-------|-----|
| Navy | `#0B3A5E` |
| Bright Blue | `#1E73BE` |
| Gold / Yellow | `#FFC72C` |
| Coal | `#374151` |
| Gray | `#8E97A3` |
| White | `#FFFFFF` |

### Usage Guidance

**Navy**
- primary headers
- outlines
- logo text
- navigation bars

**Bright Blue**
- AI emphasis
- buttons
- highlights
- link colors

**Gold / Yellow**
- accent moments
- CTA highlights
- underline accents
- folder visuals

**Coal**
- UI dark backgrounds
- strong neutral surfaces

**Gray**
- secondary surfaces
- muted UI text

**White**
- primary background
- breathing space

### Recommended UI Color Ratio

- 50% white / light neutral
- 25% navy / coal
- 15% bright blue
- 10% yellow accents

Yellow should remain an **accent color**.
