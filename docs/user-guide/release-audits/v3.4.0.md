## Summary

Version 3.4.0 introduces breaking changes to GC compaction behavior and adds comprehensive CLI testing infrastructure.

## Changes Audited

### Breaking Changes
- [x] GC compaction now preserves plan YAML to `.grabby/history/` instead of deleting
- [x] Users upgrading from 2.x will see plan files accumulate in history directory
- [x] Migration path: No action required, existing archives remain compatible

### New Features
- [x] Calculator CLI test app validates full Grabby workflow
- [x] Plan YAML preservation ensures execution context is retained
- [x] Robust metrics ID extraction using `extractContractId`
- [x] `feature gc archive` and `feature close` allowed in externalLlmOnly mode

### Bug Fixes
- [x] Defensive check in `addToHistory` prevents crash on malformed history files
- [x] Metrics gracefully handles malformed contracts during ID extraction
