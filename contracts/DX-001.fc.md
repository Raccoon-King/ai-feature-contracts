# FC: Extract ensureDir and filesystem utilities
**ID:** DX-001 | **Status:** draft
**Type:** refactor

## Ticket
- Who: Grabby maintainers and contributors
- What: Extract repeated filesystem patterns into reusable utilities
- Why: 340+ repeated `if (!exists) mkdir` patterns cause maintenance burden

## Objective
Create `lib/fs-utils.cjs` with consolidated filesystem utilities to eliminate duplicated I/O patterns across the codebase.

## Scope
- Extract `ensureDir(path)` - create directory if not exists
- Extract `readJsonSafe(path, defaultValue)` - read JSON with fallback
- Extract `writeJsonAtomic(path, data)` - atomic JSON write
- Extract `readYamlSafe(path, defaultValue)` - read YAML with fallback
- Replace usages across lib/*.cjs files

## Non-Goals
- No async versions in this contract (separate DX-002)
- No changes to API server (separate contract)

## Directories
**Allowed:** `lib/`, `tests/`
**Restricted:** `node_modules/`, `.git/`
