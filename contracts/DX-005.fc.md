# FC: Add JSDoc to exported functions
**ID:** DX-005 | **Status:** draft
**Type:** docs

## Ticket
- Who: Grabby contributors and IDE users
- What: Add JSDoc documentation to all exported functions
- Why: No documentation on 44+ functions makes code hard to understand

## Objective
Add JSDoc comments to all exported functions in core lib files to improve IDE intellisense and maintainability.

## Scope
- Add JSDoc with @param, @returns, @throws to exported functions
- Document complex regex patterns with inline comments
- Add @example for commonly used functions
- Priority files: commands.cjs, features.cjs, core.cjs, config.cjs

## Non-Goals
- No behavior changes
- No TypeScript conversion
- No private function documentation (focus on exports)

## Directories
**Allowed:** `lib/`
**Restricted:** `tests/`, `node_modules/`
